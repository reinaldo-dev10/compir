# Projeto Extensionista - Raízes em Movimento

Projeto desenvolvido em HTML, CSS e JavaScript como portfólio extensionista do curso de Engenharia de Software.

Objetivo:
Promover a valorização da cultura negra e ampliar a divulgação das ações culturais no município de Aparecida/SP.

Tecnologias utilizadas:
- HTML5
- CSS3
- JavaScript
